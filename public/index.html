Available Reports

Finance Reports

  • Payments Made

    This report lists payments that were made during the given time period (previous calendar month by default). These should generally correspond to payment lines on an order record.

  • Payments for Shipping

    This report lists payment transactions that were made for shipping charges during a given time period (previous calendar month by default). This query searches the transactions table for items with the fund code zshp*.

Cataloging Reports

  • Orphaned Subject Headings

    This report lists subjects headings that are used by at least one record in the catalog and do not have a corresponding authority record. The text has been formatted for batch use in OCLC.

Circulation Reports

  • Duplicate Patrons

    This report lists patrons who have two or more patron records in Sierra with the same eID.
